Colorschemes

You can configure the colorscheme you like by setting scad.colorscheme to appropritate string:

fig, axs = plt.subplots(5,2, figsize=(12,24)) # 2,5
axs = axs.flatten()

for ax, cs in zip(axs, colorschemes):
    ax.imshow(view(d, 300, 300, cs))
    ax.axis("off")
    ax.set_title(cs)
